# University Grading System

A simple web-based University Grading System that calculates the total marks, average, and grade for 5 subjects.

## Features

- Input marks for 5 subjects (0-100)
- Calculates total marks out of 500
- Calculates average percentage
- Displays the corresponding grade:
  - A+ (90-100)
  - A  (80-89)
  - B+ (70-79)
  - B  (60-69)
  - C  (50-59)
  - D  (40-49)
  - F  (Below 40)

## Technologies Used

- HTML
- CSS
- JavaScript

## Usage

1. Clone or download this repository:
   ```bash
   git clone https://github.com/laviee143/university-grading-system.git
2. Open index.html in your web browser.

3. Enter the marks for each subject.

4. Click Calculate Grade to see the results.
